Simple first pass - plugin for feedBack project to automate scanning in gp5 files.

Point it at the directory, it creates a midi audio track.  It takes a while.  Its 'not particularly good'.  Tough.  

Someone wants to do better with automating pulling in proper audio files from some source, be my guest.

Install:
Manual
 - Create a folder in your plugin directory 'gp-import'
 - Ex: /home/<user>/.config/feedback-desktop/plugins/gp-import
 - Put the json, .py, html, and .js files into this folder
 - Launch game, go under plugins, find the gp-import plugin, and import

"Automated"
 - Download the .zip
 - Extract the zip in your plugin directory
 - See above
